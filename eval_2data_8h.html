◆ lst_insert_elem()

ResNode * lst_insert_elem ( ResNode *  lst,
int  index,
int  side,
ResNode *  elem 
)

Inserta un elemento en la posición indicada de una lista.

Parámetros
lstLista (nodo LST_R).
indexPosición de inserción. Puede ser negativa.
side-1 o 1. Ver descripción detallada.
elemElemento a insertar.
Devuelve
NULL o ResErrorN.

Un index mayor al último índice equivale al último indice.

side indica el lado de inserción respecto al elemento existente ubicado en index.

  • side == -1: insertar a la izquierda del elemento en index. El nuevo elemento pasa a tener índice index.
  • side == 1: insertar a la derecha del elemento en index. El nuevo elemento pasa a tener índice index + 1.
